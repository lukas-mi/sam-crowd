There are two modes that you can switch between:

  • mode for annotating the components of an argument via text highlighting.

    Once mode is selected you can highlight the text of an article, which causes annotation widget to appear. In annotation widget you need to specify the tag of the component. In the widget there is auto-completion, so when you start typing the name of a tag suggestions will appear and you can choose one of them. Finally, click OK button.

    In order to delete the highlighted component, Components mode needs to be active. Then, you select the already highlighted text and in the widget click icon.

    To change the tag of the highlighted text, Components mode needs to be active. Then, you select the already highlighted text and in the widget click on the current tag. The current tag will then expand, showing an , which you click to remove the tag. Lastly, add the new tag the same way as specified previously.

  • mode for identifying relations between Premise and other components (Premise/ClaimFor/ClaimAgainst) via link drawing.

    Once mode is selected you can draw links between the components. First, click on a highlighted text area that corresponds to a Premise component. Next, move your cursor to another component and click on it (you should see a dashed link being drawn). Once the second component is clicked, a widget will appear where a relation tag (Attack/Support) needs to be specified similarly to component tag. Lastly, click icon.

    To delete the link between two components, Relations mode needs to be active. Select the relation tag on the link and then in the widget click icon.

    In order to change the tag of a link, Relations mode needs to be active. Select the relation tag on the link, in the widget delete the old tag text, type the new one and click icon.

For convenience, annotation mode will change automatically to Relations when a Premise component is highlighted. Similarly, it will change back to Components mode once a link is created. You can of course change the mode manually by toggling the switch.

When you are satisfied with your annotations click Submit button in the bottom left corner to finish the main annotation task.
